        NAME
            chown - change file owner

        SYNOPSIS
            chown <user> <path>

        DESCRIPTION
            Change the owner of a file or directory. Only root can
            use this command.

        EXAMPLES
            chown alice /home/alice/file.txt

        SEE ALSO
            chmod, stat
